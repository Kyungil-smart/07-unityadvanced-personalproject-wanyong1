# Baba Is You(모작)

## 게임 소개

이 프로젝트는 퍼즐 게임 **Baba Is You**의 핵심 규칙 시스템을 Unity로 구현한 개인 프로젝트입니다.

게임 내에서 **텍스트 블록이 곧 게임의 규칙**이 됩니다.  
플레이어는 텍스트 블록을 밀어 문장을 만들고, 그 문장에 따라 게임의 규칙이 실시간으로 변경됩니다.

예를 들어

- **BABA IS YOU** → Baba를 조작할 수 있음
- **FLAG IS WIN** → Flag에 닿으면 승리

텍스트 블록의 위치를 바꾸면 규칙이 바뀌기 때문에 이를 활용하여 퍼즐을 해결해야 합니다.

---

## 조작 방법

| 키     | 기능                      |
| ------ | ------------------------- |
| 방향키 | 캐릭터 이동               |
| Z      | 이전 행동 되돌리기 (Undo) |
| R      | 현재 스테이지 다시 시작   |
| ESC    | 일시정지 메뉴             |
